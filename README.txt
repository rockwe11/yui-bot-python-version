Для начала использования бота, необходимо добавить его на ваш сервер. Для этого необходимо создать Discord бота
через Discord Developer Portal и получить токен, который будет использоваться для авторизации бота при соединении
с Discord API.
Токен необходимо записать в файл с названием ".env", который должен выглядеть примерно так:
TOKEN=ВАШ_ТОКЕН
VK_TOKEN=ВАШ_ТОКЕН_ВКОНТАКТЕ
GROUP_ID=ID_ГРУППЫ_ВКОНТАКТЕ
PREFIX=$

Если все настроено правильно, в консоль выведется следующее сообщение:
name#tag is ready

Бот поддерживает следующие команды:
ping
help
sapper
coin
bones
balance
bonus

При использовании пользователем одной из команд, у него создается внутриигровой профиль, в начале ему дается 10000 монет.
После этого пользователю будет начислятся по 5 монет за каждое сообщение.

Чтобы посмотреть список всех команд, нужно написать команду "help"
Для просмотра информации о каждой команде по-отдельности: "help [название команды]"
